# Expense Tracker
This is a simple Expense Tracker project built using Python. It helps users record their expenses and calculates the total amount.
- Add multiple expenses
- Calculate total expenses automatically
- Handle invalid inputs
- Exit the program using "quit"
## How to Use
1. Run the Python program.
2. Enter your expense amounts one by one.
3. Type `quit` when you are finished.
4. The program will display the total amount spent.
## Example
Enter expense: 100
Enter expense: 50
Enter expense: 20
Enter expense: quit
Final Total: 170
